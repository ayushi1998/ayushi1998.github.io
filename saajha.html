Saajha DataAnalytics Dashboard

Project Scope:

  • My Role: User research, product thinking, system design
  • Tools: HTML, CSS, Bootstrap, SBAdmin, Django, MongoDB
  • Timeline: Aug 2020- Dec 2020
  • Deliverables: Requirement Gathering, Prototyping, Design, Implementation
  • Team: 4
  • My Certificate: View Certificate

About

This project was implemented in the course of Software Engineering and Open source software practices, taken by Dr. Pankaj Jalote, IIIT Delhi, India. We worked with our client - Saajha , a NGO in the field of education technology that reaches out to low-literate, government school children with school information through Whatsapp Bot.
Please view the report - Report of the Project and a short demo of the dashboard - View Dashboard

Lessons Learned:

  • Implementing good practices of front-end development like using Jinja Templating.
  • Find many open source tools, libraries like Chart.js, Table.js to help us in implementing many features of the dashboard
  • Implementing open-source software practcies while developing the entire project in the team of 4. We collaborated on GitHub We raised issues, prioritised them, assigned them to members, made pull-requests and held meetings to resolve conflicts etc.
